# Demo car owner management
- Back-end: java spring boot web-api

- Front-end: react

# How to start car demo
## Build and start web-api

$ cd web-api/

$ ./gradlew clean build

$ ./gradlew bootRun && cd ..

## Build and start front-end

$ cd front-end/

$ cp example.env .env

$ yarn install

$ yarn start && cd ..
